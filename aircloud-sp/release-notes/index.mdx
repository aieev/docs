## April 2026

### External API: New endpoints added

Added new External API endpoints for programmatic endpoint management:

- **List Endpoints**: Retrieve paginated list of accessible endpoints.
- **List Replicas**: View live replica status for active endpoints.
- **List/Get Log Files**: Access endpoint log files and their contents.
- **Patch Endpoint**: Update runtime settings for inactive endpoints.
- **Get API Key Context**: Verify API key authentication information.

## January 2026

### External API: Initial release

Launched the AirCloud External API for programmatic endpoint management:

- **Get Endpoint Status**: Check current endpoint status.
- **Start/Stop Endpoint**: Start or stop endpoints via API.
- **Scale Replicas**: Adjust replica count for active endpoints.
