## 2026년 4월

### External API: 신규 엔드포인트 추가

프로그래밍 방식의 엔드포인트 관리를 위한 External API 엔드포인트가 추가되었습니다:

- **엔드포인트 목록 조회**: 접근 가능한 엔드포인트 목록을 페이지네이션으로 조회.
- **레플리카 목록 조회**: 활성 엔드포인트의 실행 중인 레플리카 상태 조회.
- **로그 파일 목록/내용 조회**: 엔드포인트 로그 파일 및 내용 접근.
- **엔드포인트 설정 수정**: 비활성 엔드포인트의 런타임 설정 변경.
- **인증 컨텍스트 조회**: API Key 인증 정보 확인.

## 2026년 1월

### External API: 최초 출시

프로그래밍 방식의 엔드포인트 관리를 위한 AirCloud External API가 출시되었습니다:

- **엔드포인트 상태 조회**: 현재 엔드포인트 상태 확인.
- **엔드포인트 시작/중지**: API를 통한 엔드포인트 시작 및 중지.
- **레플리카 스케일링**: 활성 엔드포인트의 레플리카 수 조정.
