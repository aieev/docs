You can open a terminal session directly inside a running container from the AirCloud console, without any SSH setup.

## How to open

### 1. Go to the container detail page

From the project, click on the running container you want to access.

### 2. Click Web Terminal

Click **Web Terminal** in the top action bar.

### 3. Select a replica

You can select which replica to connect to. If the replica is replaced or restarted during the session, the terminal will automatically attempt to reconnect.

### 4. Start your session

A terminal session opens in the browser. You can run commands, inspect files, and debug directly inside the container.

## Notes

- The web terminal connects to a running container. It is not available for containers in Pending, Deploy Failed, or Unhealthy states.
- The session is not persistent. If the browser tab is closed, the session ends.
- For persistent connections, use [SSH access](/docs/air-container/connect-with-ssh) instead.
