AirCloud delivers notifications in real time, so you can stay informed without refreshing the page. All notifications are available from a centralized notification center.

## Key Features

<CardGroup cols={1}>
  <Card title="Real-Time Notifications" icon="zap">
    Receive important events instantly, including deployment status changes and platform announcements.
  </Card>

  <Card title="Notification Center" icon="inbox">
    View your notification history in one place, browse older notifications, and manage them easily.
  </Card>

  <Card title="Credit and Usage Alerts" icon="circle-alert">
    Receive alerts for low credits, GPU usage charges, and payment failures so you can take action before your services are interrupted.
  </Card>
</CardGroup>

## Notification Channels

AirCloud provides two ways to receive notifications.

| Channel | Description |
| --- | --- |
| **Notification Center** | View current and past notifications by clicking the notification icon in the Console header. |
| **Desktop Notifications** | Receive browser notifications even when you are not actively viewing the AirCloud Console. |
