If you need help while using AirCloud, have technical questions, or want to request a model, please reach out through the channels below.

## What can you contact us about?

- Air API usage
- Air Container deployment issues
- Model addition requests
- Adoption and technical consulting
- Onboarding and organization setup
- Other technical inquiries

## Information to include

Including the following information helps us respond faster.

- Organization or Project name
- Feature in use (e.g., Air API, Air Container)
- Description of the issue
- Steps to reproduce
- Screenshots or log information (if available)

## Contact

For adoption consulting or technical support, please reach out below.

## Feedback and model requests

If you need a model that is not currently supported, or have product improvement suggestions, please include the following details.

- Model name
- Intended use case
- Expected scale or traffic characteristics
- Preferred response speed or operating model
