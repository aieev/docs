Air Container 배포 시 자주 사용하는 런타임 설정 항목을 정리한 문서입니다.

## 지원 레지스트리 예시

- GCP Artifact Registry
- AWS ECR
- GitHub Container Registry
- Docker Hub

<Note>
  비공개 레지스트리를 사용하는 경우에는 인증 정보가 필요합니다.
</Note>

## 자주 사용하는 설정 항목

### 시작 명령

컨테이너 실행 시 사용할 시작 명령입니다.

### 포트

서비스가 외부 요청을 받을 포트입니다.

### 헬스 체크 URL

컨테이너가 정상 동작 중인지 확인하기 위한 경로입니다.

### 최대 대기 시간

시작 과정에서 기다릴 최대 시간입니다.

### 부팅 대기 시간

서비스가 준비될 때까지 허용할 대기 시간입니다.

### 환경 변수

애플리케이션 실행에 필요한 환경 변수입니다.

### 엔드포인트 ID

필요한 경우 엔드포인트 식별용으로 사용할 수 있는 값입니다.

## 운영 권장 사항

- 공개 레지스트리와 비공개 레지스트리 설정을 구분해 관리
- 포트와 헬스 체크 경로를 배포 전 미리 검증
- 운영 환경 변수는 배포 전 다시 확인
- 설정 변경 후에는 로그와 Usage 상태를 함께 점검
