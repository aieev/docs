시크릿 키를 발급받은 뒤 실제 애플리케이션에서 API를 호출하는 방법을 안내합니다. Air API는 OpenAI 호환 인터페이스를 제공하므로, OpenAI SDK를 사용하는 기존 코드에서 엔드포인트와 키만 교체해 바로 연동할 수 있습니다.

## 엔드포인트와 시크릿 키 확인

콘솔에서 API 워크로드를 선택한 뒤 **코드 보기** 버튼을 클릭하면 해당 모델의 엔드포인트 URL과 코드 샘플을 확인할 수 있습니다.

![콘솔 코드 보기](/images/air-api/console-code-view.png)

이 페이지의 예시 코드에서 `YOUR_ENDPOINT`와 `YOUR_API_KEY`는 콘솔에서 확인한 값으로 교체하세요.

## 인증

모든 API 요청에는 `Authorization` 헤더에 시크릿 키를 포함해야 합니다.

```
Authorization: Bearer YOUR_API_KEY
```

<Warning>
  시크릿 키는 생성 시 한 번만 확인할 수 있습니다. 키는 코드에 직접 하드코딩하지 않고 환경 변수로 관리하는 것을 권장합니다. 분실한 경우 콘솔에서 재발급할 수 있습니다.
</Warning>

## 코드 예시

### Chat 모델

<CodeGroup>
```bash curl
curl YOUR_ENDPOINT/chat/completions \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "YOUR_MODEL_ID",
    "messages": [
      {"role": "user", "content": "안녕하세요"}
    ]
  }'
```

```python Python
from openai import OpenAI

client = OpenAI(
    api_key="YOUR_API_KEY",
    base_url="YOUR_ENDPOINT"
)

response = client.chat.completions.create(
    model="YOUR_MODEL_ID",
    messages=[
        {"role": "user", "content": "안녕하세요"}
    ]
)

print(response.choices[0].message.content)
```
</CodeGroup>

### TTS 모델

<CodeGroup>
```bash curl
curl YOUR_ENDPOINT/audio/speech \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "YOUR_MODEL_ID",
    "input": "안녕하세요. 오늘 날씨가 좋네요.",
    "voice": "aiden"
  }' \
  --output output.mp3
```

```python Python
from openai import OpenAI

client = OpenAI(
    api_key="YOUR_API_KEY",
    base_url="YOUR_ENDPOINT"
)

response = client.audio.speech.create(
    model="YOUR_MODEL_ID",
    input="안녕하세요. 오늘 날씨가 좋네요.",
    voice="aiden"
)

response.stream_to_file("output.mp3")
```
</CodeGroup>

### Embedding 모델

<CodeGroup>
```bash curl
curl YOUR_ENDPOINT/embeddings \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "YOUR_MODEL_ID",
    "input": "텍스트를 벡터로 변환합니다."
  }'
```

```python Python
from openai import OpenAI

client = OpenAI(
    api_key="YOUR_API_KEY",
    base_url="YOUR_ENDPOINT"
)

response = client.embeddings.create(
    model="YOUR_MODEL_ID",
    input="텍스트를 벡터로 변환합니다."
)

print(response.data[0].embedding)
```
</CodeGroup>

## 운영 시 주의사항

- **키 보안**: 시크릿 키는 환경 변수로 관리하고, 코드 저장소에 커밋하지 마세요.
- **사용량 확인**: 콘솔의 사용량 탭에서 모델별 호출량과 비용을 정기적으로 확인하세요.
- **크레딧 부족**: 크레딧이 소진되면 API 호출이 중단됩니다. 자동 충전을 설정해두는 것을 권장합니다.
