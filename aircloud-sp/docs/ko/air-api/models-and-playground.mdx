Air API에서는 프로젝트 안에서 지원 모델을 선택하고, Playground를 통해 브라우저에서 바로 모델 응답을 테스트할 수 있습니다.

## 지원 모델

<Info>
  개별 모델의 상세 스펙과 사용 예시는 [모델 탭](/models/ko)에서 확인할 수 있습니다.
</Info>

| 모델 이름 | 타입 |
| :-: | :-: |
| Qwen: Qwen3-Embedding-8B | Embedding |
| Qwen: Qwen3-TTS | TTS |
| Qwen: Qwen3.5-9B | Chat |
| Qwen: Qwen3.5-35B-A3B | Chat |

- **Chat**: 텍스트 입력에 대한 응답을 생성하는 대화형 모델입니다. 일반 대화, 요약, 코드 생성 등에 활용할 수 있으며, 입력/출력 토큰 수(/M tokens) 기준으로 과금됩니다.
- **Embedding**: 텍스트를 벡터로 변환하는 모델입니다. 검색, 분류, 유사도 계산 등에 활용할 수 있으며, 입력 토큰 수(/M tokens) 기준으로 과금됩니다.
- **TTS**: 텍스트를 음성으로 변환하는 모델입니다. 문자 수(/K chars) 기준으로 과금됩니다.

필요한 모델이 목록에 없는 경우 피드백 채널을 통해 추가를 요청할 수 있습니다.

## Playground

Playground는 브라우저에서 바로 모델을 테스트할 수 있는 Web UI입니다. 애플리케이션 연동 전에 모델 동작과 파라미터를 빠르게 확인하는 데 유용합니다.

<Note>
  Playground는 **Chat 및 TTS 모델**에서만 지원합니다. Embedding 모델은 Playground를 제공하지 않으며, API 호출로만 사용할 수 있습니다.
</Note>

Playground 상단에는 두 개의 탭이 있습니다.

- **Playground**: 인터페이스에서 직접 모델을 테스트합니다.
- **README**: 선택한 모델의 사용법과 특성을 확인합니다.

우측 상단 버튼으로 **API 키 발급하기**와 **코드 보기**에 바로 접근할 수 있습니다.

모델 타입에 따라 Playground의 인터페이스와 설정 항목이 다릅니다.

### Chat 모델

Chat 모델 Playground는 채팅 인터페이스로 제공됩니다. 텍스트를 입력하면 모델 응답을 바로 확인할 수 있습니다.

![Chat 모델 Playground](/images/air-api/playground-chat.png)

오른쪽 **채팅 설정** 패널에서 모델 동작에 영향을 주는 파라미터를 조정할 수 있습니다.

| 설정 | 설명 |
| :-- | :-- |
| **시스템 프롬프트** | 모델에게 역할이나 행동 지침을 부여하는 초기 지시문. 대화 전체에 걸쳐 적용됩니다. |
| **추론(Reasoning)** | 답변 전 내부 추론 과정을 수행하도록 활성화합니다. 복잡한 문제에서 정확도가 향상될 수 있습니다. |
| **Temperature** | 응답의 무작위성을 조정합니다. 높을수록 다양하고 창의적인 응답, 낮을수록 일관되고 예측 가능한 응답이 생성됩니다. |
| **최대 토큰** | 한 번 응답에서 생성할 수 있는 최대 토큰 수를 설정합니다. |
| **Top P** | 확률 상위 P% 이내의 토큰만 후보로 사용합니다. Temperature와 함께 응답 다양성을 제어합니다. |
| **빈도 페널티** | 이미 등장한 토큰에 페널티를 부여해 반복 표현을 줄입니다. |
| **존재 페널티** | 한 번이라도 등장한 토큰 전반에 페널티를 부여해 새로운 주제로 유도합니다. |

### TTS 모델

TTS 모델 Playground는 텍스트를 입력하면 음성 파일을 생성하는 인터페이스로 제공됩니다. 음성으로 변환할 텍스트를 입력하고 **생성** 버튼을 클릭하면 오디오를 바로 재생할 수 있습니다.

![TTS 모델 Playground](/images/air-api/playground-tts.png)

오른쪽 **TTS 설정** 패널에서 음성 생성 방식을 조정할 수 있습니다.

| 설정 | 설명 |
| :-- | :-- |
| **지시사항** | 음성 톤이나 말투에 대한 지침을 입력합니다. (예: "친근하고 매력적인 톤으로 말하세요") |
| **음성** | 사용할 목소리를 선택합니다. (예: Aiden) |
| **속도** | 음성 재생 속도를 조정합니다. 기본값은 1×입니다. |
| **응답 형식** | 생성될 오디오 파일 형식을 선택합니다. (예: MP3) |

<Tip>
  Playground 상단에는 현재 선택된 모델의 가격이 표시됩니다. Chat/Embedding 모델은 ₩/M tokens, TTS 모델은 ₩/K chars 기준이며, 통화 토글(₩ / \$)로 원화와 달러 기준을 전환할 수 있습니다.
</Tip>
