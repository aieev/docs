Air API 사용 중 자주 발생하는 문제와 기본 확인 항목을 정리한 문서입니다.

## 시크릿 키를 잃어버린 경우

시크릿 키는 생성 이후 다시 확인할 수 없습니다. 기존 키를 확인할 수 없는 경우에는 새 키를 발급받아 교체해야 합니다.

## 모델이 보이지 않는 경우

프로젝트와 Air API 화면을 다시 확인합니다. 필요한 모델이 아직 제공되지 않는 경우에는 피드백 채널을 통해 요청할 수 있습니다.

## 코드 샘플 테스트가 실패하는 경우

아래 항목을 우선 확인합니다.

- 올바른 프로젝트를 선택했는지
- 시크릿 키를 정확히 입력했는지
- 잘못된 키를 사용하고 있지 않은지
- 크레딧 또는 과금 상태에 문제가 없는지

## Playground 테스트가 되지 않는 경우

브라우저 새로고침 후 다시 시도하고, 프로젝트 선택 상태와 모델 선택 상태를 다시 확인합니다.

## 지원이 필요한 경우

문제가 계속되면 [문의 및 피드백](/docs/ko/support/contact-and-feedback) 문서를 통해 지원을 요청하시기 바랍니다.
