AirCloud 사용 중 도움이 필요하거나, 기술 문의 또는 모델 요청이 필요한 경우 아래 채널을 통해 문의할 수 있습니다.

## 어떤 내용을 문의할 수 있나요?

- Air API 사용 방법
- Air Container 배포 문제
- 모델 추가 요청
- 도입 및 기술 상담
- 온보딩 및 조직 설정 관련 문의
- 기타 기술 문의

## 문의 시 함께 보내면 좋은 정보

아래 정보를 함께 전달하면 더 빠르게 확인할 수 있습니다.

- Organization 또는 Project 이름
- 사용 중인 기능 (예: Air API, Air Container)
- 발생한 문제 상황
- 재현 단계
- 화면 캡처 또는 로그 정보 (가능 시)

## 문의처

도입 상담 및 기술 지원이 필요한 경우 아래로 연락해 주세요.

- 김현경: **hk.kim@aieev.com**
- 박병수: **bspark@aieev.com**

## 피드백 및 모델 요청

현재 지원되지 않는 모델이 필요하거나 제품 개선 의견이 있는 경우, 요청 내용을 함께 전달해 주시기 바랍니다.

특히 아래 내용은 함께 보내면 좋습니다.

- 필요한 모델명
- 예상 사용 목적
- 요청 규모 또는 트래픽 특성
- 원하는 응답 속도 또는 운영 형태
