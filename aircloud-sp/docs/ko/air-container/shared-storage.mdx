컨테이너를 재시작하거나 중지해도 데이터가 유지되는 영구 저장소입니다. 하나의 프로젝트 안에서 볼륨을 공유하여 여러 컨테이너 또는 여러 레플리카가 함께 사용할 수 있습니다.

## 영구 볼륨으로 할 수 있는 것

영구 볼륨을 배포 전반에서 더 유연하게 사용할 수 있습니다.

- **오토스케일링 지원**: 오토스케일링과 여러 레플리카를 사용하는 배포에도 볼륨을 연결할 수 있습니다.
- **생성 시 새/기존 볼륨 선택**: 배포를 생성할 때 새 볼륨을 만들거나 기존 볼륨을 재사용할 수 있습니다.
- **여러 배포에서 공유**: 하나의 볼륨을 여러 배포에 연결할 수 있으며, 볼륨 선택 시 이미 사용 중인 배포를 함께 보여줍니다.
- **중지 상태에서 변경**: 배포가 중지된 동안 연결된 볼륨을 활성화/비활성화하거나 교체할 수 있습니다.

## 사용 방법

### 1. 새 볼륨 만들기

컨테이너 생성 시 영구 볼륨을 새로 생성하여 연결할 수 있습니다. 볼륨 용량은 최대 30GB까지 설정할 수 있습니다.

![새 볼륨 만들기](/images/first_step.png)

### 2. 기존 볼륨 연결하기

다른 컨테이너에 이미 사용 중인 볼륨을 다시 연결하여 같은 볼륨을 공유할 수 있습니다.

![기존 볼륨 연결하기](/images/second_step.png)

### 3. 여러 레플리카에서 같은 볼륨 공유하기

하나의 컨테이너에서 여러 레플리카가 같은 볼륨을 함께 사용할 수 있습니다.

![여러 레플리카 볼륨 공유](/images/shared_storage.png)
