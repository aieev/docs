This page covers common problems when using Air API and the first checks to perform.

## If you lost your secret key

Secret keys cannot be viewed again after creation. If you no longer have the key, generate a new one and rotate your integration.

## If a model doesn’t appear

Double-check the selected project and the Air API screen. If the model isn’t available yet, request it via the feedback channel.

## If the code sample test fails

Check the following first:

- You selected the correct project
- You pasted the secret key correctly
- You are not using an old/invalid key
- Your credits/billing status is valid

## If the Playground doesn’t work

Refresh the browser and verify the project and model selection state.

## If you need support

If the issue persists, request help via [Contact & Feedback](/docs/support/contact-and-feedback).
